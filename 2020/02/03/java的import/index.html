java的import

创建时间:2020-02-03 17:36
阅读:
评论:
  1. Package
  2. import

Package

  • 如果不是放在包下的类不需要进行声明，不然需要在第一行需要声明包名
  • 包名唯一，通常使用域名逆序来设置包名
  • 引入类时必须使用类的完整名字即包名+类名

import

  • import 放在package下面和类定义的上方
  • 可以用*来引用一个目录下的所有类，比如import java lang.*,但是*只包含当前目录下的java文件不包含子文件和子文件夹的java文件，由于使用*可能会带来冲突所以尽可能减少使用
