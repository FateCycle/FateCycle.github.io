java的Set

set

HashSet 无序，不支持同步，基于散列函数的集合

TreeSet 可排序，不支持同步，基于树结构的集合

LinkedHashSet 可排序，不支持同步，基于散列函数和双向链表集合

HashSet主要api:

  • size 获取集合中元素个数

  • add 添加一个元素

  • clear 清楚整个hashSet

  • contains 是否包含某个元素

  • remove 删除一个元素

  • retainAll 计算两个集合的交集，如:a.retainAll(c)那么a会将和c共有的元素保留下来其余都进行删除

LinkedHashSet同上唯一的不同是LinkedHashSet会保留add时的顺序而HashSet不会保留原来添加进来的顺序

TreeSet 和上面两个不同他不能容纳null元素，并且TreeSet的排序是由元素的compareTo方法或者Comparator来进行的

注意事项:

  • set的所有实现类都是不同步的想要同步需要调用Set s = Collections.synchronizeSet(new LinkedHashSet())

  • 三个Set实现类的元素都只能是对象所以不能用基本类型放到集合而要使用包装类

  • HashSet和LinkedHashSet判断元素重复原则

    先根据元素的类中的hashCode()方法若返回相同，则继续判断equals()方法如果也相同那么这两个对象一致那么set会剔除其中一个来保证set的数据均不相同特点，如果这两个方法有一个返回不同的值那么程序都会认为新增的对象和已有集合的对象没有重复

  • TreeSet判断重复元素是更具compareTo方法所以TreeSet的元素对应的类必须继承Comparable接口

    
package xXX;

public class Cat implements Comparable{
    String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public Cat(String name) {
        super();
        this.name = name;
    }

    @Override
    public String toString() {
        return "Cat [name=" + name + "]";
    }

    @Override
    public int compareTo(Object arg0) {
        // TODO Auto-generated method stub
        return name.compareTo(((Cat)arg0).getName());
    }
}
